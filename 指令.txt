專題所有指令:
	
	一、基本:
	
		1. cleanmem -> 清除記憶體		ok
		
		2. clear -> 清空畫面+記憶體		ok
		
		3. print -> 印出				ok
		
		4. 賦值 -> 通常包含 '=' 的指令	ok
		
		5. 創建新矩陣 -> 通常包含'[]'	ok
		
	二、運算:
		
		1. 2 +-*/ 3 -> 數值加減乘除
		
		2. [] +/* [] -> 矩陣加減乘
		
		3. a +-*/ b -> 變數加減乘除(變數可為任意型態，但矩陣沒有除)
		
	三、不同型態特有函式:
	
		矩陣:
			
			1. .T -> 轉置
			
			2. .I -> 反矩陣
			
			3. .D -> 行列式
		
		多項式:
		
			1. .dif -> 微分
			
			2. .S -> 不定積分 or 定積分(根據S函式內放的參數個數判斷)
			
			3. .lim -> 取極限
